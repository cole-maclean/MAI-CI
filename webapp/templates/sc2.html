Select your race and enemy race.
The next build unit is given when a current unit is clicked on. Custom builds can be made by inputting the desired unit and player (ie. SCV0, Marine1 etc.) and clicking on the node to insert that unit. This will overwrite the clicked unit and delete all units that follow where the custom unit is inserted. Longer custom builds can be inserted using comma delimeted units (ie. SCV0,Marine1). Clicking the Auto Build button will add the next Auto Build Length number of units to the current build order.
